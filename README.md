🌾 KisaanSaathi – Smart Farming Platform

A modern AI-powered agriculture platform designed to help farmers make smarter farming decisions.
KisaanSaathi provides tools like weather updates, crop health detection, mandi prices, and farmer profile management in one easy-to-use dashboard.

🚀 Features

🌦 Weather Information

Real-time weather updates for farmers

Temperature, humidity, and wind data

🌱 Crop Health Detection

Upload crop images to analyze plant health

Helps detect possible crop diseases

💰 Mandi Price Checker

View crop prices by state and market

Helps farmers sell crops at the best price

👤 Farmer Profile Management

Store farmer details like name, crops, and location

🌙 Dark / Light Mode

Modern UI with theme switching

📊 Farmer Dashboard

Quick overview of weather, crop alerts, and mandi prices
